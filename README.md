# 🎥 AI Creative StoryTeller

Create cinematic stories with AI using this futuristic scroll-based web interface.

## ✨ Features

- 🌌 **Cinematic UI**: Powered by TailwindCSS and beautiful static glowing Glassmorphism backgrounds.
- 🎬 **AI Movie Director**: Provide a prompt and watch the AI generate scenes in real time. Features a Creative Director agent that intelligently interleaves text, images, and audio into fluid, mixed-media experiences (Interactive storybooks, marketing assets, explainers).
- 🛡️ **Resilient Backend System**: Utilizes multiple Google Gemini API keys gracefully failing over natively for ultimate uptime.
- 🔊 **Audio & Visual Integration**: Automatically plays AI-generated narration alongside generated storyboards.
- 🪟 **Responsive Layout**: Sleek dashboards that look great on any device size.
